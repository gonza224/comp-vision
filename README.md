# Repository for CSC 391: Special Topics — Computer Vision

This repository contains the programming assignments for the **CSC 391: Special Topics — Computer Vision** course at WFU. Each module (folder) represents a major topic covered in the course.

Geometric transforms: This is a cool experiment in which we can learn how the transformation matrices we learned in class can be created and used easily with OpenCV.
Lens aperture: This experiment helps us visualize how lens to image distance and aperture diameter change with focal length, showing how the thin lens law applies to real camera parameters.
Sampling quantization: This experiment allows us to observe how a smooth, continuous signal becomes digital when sampled at different sampling frequencies.
Error noise: This experiment shows how noise affects real world signals by adding random Gaussian noise, then measuring the resulting error using MSE, RMSE, and PSNR.
